Galeria Z-Masters

JavaScript

Lenguaje que hace interactivas las
páginas web y es fundamental
en el desarrollo front-end.

SCSS

Una sintaxis de Sass, un preprocesador
que organiza y extiende las capacidades
del CSS tradicional.

HTML

El lenguaje de marcado estándar utilizado para estructurar y presentar el contenido de la web.

Angular

Framework de código abierto de Google
para construir aplicaciones web
complejas de una sola página (SPA).

PHP

Lenguaje de scripting popular de lado del servidor diseñado principalmente para el desarrollo web dinámico.

Node.js

Entorno de ejecución de JavaScript para
el lado del servidor, ideal para construir aplicaciones de red rápidas.
